As of now, I'm not ready to share the PS creator.
My Javascript-fu is pretty weak, so it's taking me a while to clean it up enough to be shared.
But here's how the creator works:
  • Generate IV and Salt (using browser cryptography APIs)
  • Input password hint
  • Input password
  • Input plaintext (or open file/image from disk)
  • Upon hitting save, a Portable Secret (HTML file with encrypted payload) is generated and downloaded to disk
Encryption works offline, is dependency free and is carried out entirely in the browser window context (just like decrypt). This example is an example output.
